# MaidConnect API — Index

This folder contains expanded API documentation and endpoint scaffolds derived from the Postman collection and `documents.conf`.

Files
- `expanded-endpoints.md` — full list of 300 endpoint entries (compact request/response examples).
- `auth.md` — auth-related endpoints summary.
- `workers.md` — worker profile steps, documents, verification, guarantees.
- `admin.md` — admin endpoints: verifications, audits, reports, settings.

Use this as the starting point for generating an OpenAPI spec or Postman collection.
